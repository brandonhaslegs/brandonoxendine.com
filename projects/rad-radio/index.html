RAD RADIO

Rad Radio is a decentralized, live-streamed events platform made possible by the decentralized streaming platform LivePeer, Radicle. While the broadcast is streaming viewers can support the DJs using Radicle Drips, an Ethereum-based money streaming protocol.

Role: Naming, Visual Design, Branding, Code

The lineup was published the day before the event. There was a countdown timer on the lineup that counted down to the start time.

The video was live streamed on the actual website as a full-screen background.

Users could support the DJs using Radicle Drips and receive an NFT in their wallet as a sort of keepsake of the event.
